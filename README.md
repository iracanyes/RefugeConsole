# Refuge - Application .NET Console

## Description
L'objectif du programme est la gestion d’un refuge d’animaux composés de chiens et de chats

## Installation et démarrage

1. Télécharger le project 
2. Copier les fichiers ``.env.dist`` et ``RefugeConsole/.env.dist`` vers leurs fichiers d'environnement correspondant ``.env`` et modifier toutes les valeurs requises
3. Lancer la construction de containers  Docker avec Docker-Compose, lancer la commande ``docker-compose up -d `` à la racine du projet
4. Utiliser le fichier ``data/creertables_1.sql`` pour créer les tables en base de donnée
5. Démarrer l'application
